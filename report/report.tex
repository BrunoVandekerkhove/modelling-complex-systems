\fakesubsection{Custom constants}

I added 2 constants ; one called \texttt{first\_pass} (to make it easier to enforce that every first pass is unique) and one called \texttt{tracks\_undirected} making it easier to encode some axiom enforcing symmetry of tracks such that it doesn't have to be done manually.
